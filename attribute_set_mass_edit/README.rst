This module allows the user to authorize mass editing on an attribute.

**Table of contents**

.. contents::
   :local:

Usage
=====

On the attribute, thick the check box 'Allow Mass Editing' and refresh your browser.
A Mass Editing profile is automatically created per attribute group.
